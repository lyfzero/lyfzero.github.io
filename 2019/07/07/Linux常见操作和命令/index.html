实用程序

文件和文件夹操作

  1. 创建/删除目录mkdir/rmdir

  2. 创建/删除文件touch/rm

    • touch创建空白文件或修改文件时间
      • 更改内容的时间（mtime），更改权限的时间（ctime），最后访问的时间（atime）
    • rm的参数（可合并）
      • -i：交互式删除（每次删除都要询问）
      • -r：删除目录并递归删除目录中的文件和目录
      • -f：强制删除，没有提示

  3. 切换和查看当前工作目录

    • cd：切换，可跟相对路经或绝对路径，可用cd ..返回上级目录
    • pwd：查看当前目录

  4. 查看目录内容：ls

    • -l：以长格式查看文件和目录
    • -a：显示隐藏文件
    • -R：递归展开目录（可怕）
    • -d：只显示目录
    • -t/-S：按时间/大小排序

  5. 查看文件内容：cat/head/tail/more/less!

  6. 拷贝/移动文件：cp/mv

  7. 查找文件/查找内容：find/grep

    pass（grep在搜索字符串时可用正则表达式）

  8. 链接：ln

    • 硬链接：指向文件数据的指针
    • 软链接：类似快捷方式

  9. 压缩/解压缩，归档/解归档：gzip/gunzip/xz/tar

    pass

  10. 其他：

    • sort：文件内容排序
    • uniq：去除相同的
    • diff
    • tr：cat file | tr '\t' ','把file中的制表符用逗号替代再cut出来
    • cat：cat file1把file1的内容展示出来
    • paste：paster file1 file2把file的内容粘贴到file2
    • wc
    • file：识别文件

文件系统

文件和路径

  1. 命名规则：

    • 最大长度（与文件系统类型有关），一般不超过255个字符（怎么会这么多。。。）
    • 最好英文大小写字母、数字、下划线、点
    • 尽量不用空格，否则输入文件名时需用“”或“\\”转义

  2. 扩展名：可选

    （没写过）

  3. 隐藏文件：以点开头的

目录结构

目录 说明 目录 说明
/bin 基本命令的二进制文件 /proc 内核和进程信息
/boot 引导加载程序的静态文件 /root 超级管理员用户主目录
/dev 设备文件 /run 存放系统运行时需要的东西
/etc 配置文件 /sbin 超级用户的二进制文件
/home 普通用户主目录的父目录 /sys 设备的伪文件系统
/lib 共享库文件 /tmp 临时文件夹
/media 自动识别设备的挂载目录 /usr 用户应用目录
/mnt 临时挂载文件系统的挂载点 /var 变量数据目录
/opt 可选插件软件包安转位置 /srv ?

访问权限

  1. chmod-改变文件模式比特

    • 字符设定
    • 数字设定

    | 文件类型 | 所有者权限 | 同组用户权限(g) | 其他用户权限(o) |
    | :———-: | :——————: | :——————-: | :——————-: |
    | d（目录） | r w x（7） | r - x（5） | r - x（5） |
    | -（文件） | r w -（6） | r - -（4） | r - -（4） |
    | l（链接） | r w x（7） | r w x（7） | r - x（5） |

  2. chown-改变文件所有者

    chown xxx(user) xxx(filename)
