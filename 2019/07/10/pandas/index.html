cheat sheet

  1. 创建pandas两种对象：

    • DataFrame：一个表，字典。包含了单独的条目，用pd.DataFrame()生成对象，键表示列的名字，值表示列下的元素，行的名字可以用index规定
    • Series：一连串的值，列表。可以用一个列表生成pd.Series()，只有整个对象的名字
    • 可以把dataframe看成是一系列series连起来的结构

  2. 从已经存在的数据中导入导出，常见的数据格式

    • CSV（comma-separated values）：一个值用逗号隔开的表
      • 导入：xxx_reviews = pd.read_csv(xxx_path)
      • 查看DataFrame的形状：xxx_reviews.shape
      • 检查（前五行）：xxx_reviews.head
      • 不要行索引：read_csv时设置index_col=0
      • 导出：xxx_reviews.head().to_csv(csv_filename)
    • Excel()
      • 导入：要指定哪一张xxx = pd.read_excel(xxx_path,sheet_name=xxxx)
      • 不一定好导入，需要提前处理。。。
      • 导出：xxx.to_excel(excel_filename, sheet_name=xxxx)
    • SQL(Structured Query Language)：一系列有名字的表
      • 创建一个connector，用sqlite3库conn = sqlite3.connect(xxx_path)
      • 导入：`xxx = pd.read_sql_query(“SELECT * FROM fires”, conn)

  3. 索引：

    • dataframe像调用属性一样xx.xxx或xx['xxx']，series：下标索引

    • 用pandas自己的索引iloc和loc范式

    • iloc：index-based selection

      xxx.iloc[0]引用一列（包括列标），xxx.iloc[:,0]（不包括列标）

    • loc：Label-based selection，在列标有意义时很有用

    • iloc 使用 Python stdlib 索引方案,其中包含范围的第一个元素,排除最后一个元素。因此 0:10 将选择条目 0,…,9。同时,包括索引。因此 0:10 将选择条目 0,…,10。

    • 操纵索引：xxx.set_index("title")

  4. 条件选择

    • e.g.reviews.loc[reviews.country == 'Italy']
    • 与：reviews.loc[(reviews.country == 'Italy') & (reviews.points >= 90)]
    • 或：`reviews.loc[(reviews.country == ‘Italy’) | (reviews.points >= 90)]``
    • 本身的：
      • `isin:reviews.loc[reviews.country.isin(['Italy', 'France'])]
      • isnull notnull

    • 赋常值
    • 赋一连串的值

    • xxx.describe()对不同类型的数据生成不同的摘要类型，生成series
    • xxx.mean()（数值型）
    • xxx.unique()
    • xxx.value_counts()，生成series

  8. grouping（分组）

    • 可对分的组进行summary function的操作

    • 可以按照多种column进行分组

    • 可以用agg进行多种函数的操作

    reviews.groupby(['country']).price.agg([len, min, max])

    • 多下标索引xxx.reset_index()下标重排

  9. 排序

    • xxx.sort_values(by=xxx)默认为升序，可设置ascending=False为降序，by可选好几列来排
    • xxx.sort_index()按下标排序

  10. 数据类型和缺失值NaN(not a number)

    • 查看数据类型xxx.dtype
    • 设置数据类型xxx.astype()

    • 补全缺失值xxx.fillna(xxxx)

  11. 重命名

    • 行标xxx.rename(index={0: xxxx, 1: xxxx})直接set_index就好
    • 重新设单独的列或行索引xxx.rename_axis(xxx,axis='rows').rename_axis(xxx, axis='columns')

  12. 组合

    • concat：pd.concat([xxx,xxx])把列标相同的组合起来
    • join：left.join(right,lsuffix='xx', rsuffix='xxx')
